## 🚀 Quick Start

### 1. Install
```bash
# Via Command Line
code --install-extension nishuuu.nishuuu-themes

# Or search "Nishuuu Themes" in VS Code Extensions
```

### 2. Activate Theme
- Press `Ctrl+K Ctrl+T` (Windows/Linux) or `Cmd+K Cmd+T` (Mac)
- Select any **"Tara"** theme from the list

### 3. Choose Your Accent
- Click the 🎨 icon in your status bar
- Pick from 14 beautiful colors or create custom ones
- Watch your theme transform instantly!

## ⚙️ Configuration

### Settings

```json
{
  "nishuuu.accentColor": "teal",
  "nishuuu.customAccentColor": "#80CBC4", 
  "nishuuu.enableStatusBar": true
}
```

| Setting | Description | Default |
|---------|-------------|---------|
| `nishuuu.accentColor` | Current accent color name | `"teal"` |
| `nishuuu.customAccentColor` | Custom hex color value | `""` |
| `nishuuu.enableStatusBar` | Show status bar color indicator | `true` |

### Commands

Access via Command Palette (`Ctrl+Shift+P`):

- **`Nishuuu: Select Accent Color`** - Open color picker
- **`Nishuuu: Toggle Status Bar`** - Show/hide status indicator
