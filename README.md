## âš™ï¸ Configuration

### Settings

```json
{
  "nishuuu.accentColor": "teal",
  "nishuuu.customAccentColor": "#80CBC4", 
  "nishuuu.enableStatusBar": true
}
```

| Setting | Description | Default |
|---------|-------------|---------|
| `nishuuu.accentColor` | Current accent color name | `"teal"` |
| `nishuuu.customAccentColor` | Custom hex color value | `""` |
| `nishuuu.enableStatusBar` | Show status bar color indicator | `true` |

### Commands

Access via Command Palette (`Ctrl+Shift+P`):

- **`Nishuuu: Select Accent Color`** - Open color picker
- **`Nishuuu: Toggle Status Bar`** - Show/hide status indicator
